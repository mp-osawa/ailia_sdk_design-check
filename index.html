エッジAI推論SDK。

組み込み・モバイル・ゲームなど、Python以外の環境でAIを動かしたい開発者向けの推論SDKです。Unity・Kotlin・C++から使える音声・LLM・トラッキングライブラリも揃っており、エッジAIアプリケーションを一気通貫で開発できます。

まずは動かして体験しよう

アイリアのデモアプリ

物体検出・姿勢推定・超解像・音声認識・音声合成・LLMなど、アイリアSDKが動かすAIをアプリで今すぐ体験できます。

エッジAI開発の現実的な課題
  • 工場・車載・現場など、通信が使えない環境でAIを動かす必要がある
  • クラウドAPIは推論ごとに費用が発生し、スケールすると費用が膨大になる
  • 画像・音声・文書などの機密データを社外に送信できない
  • UnityやKotlinからPythonのAIライブラリ（Transformers等）が使えない
アイリアSDKによる解決
  • 完全オフライン推論。ネットワーク不要でデバイス上で完結する
  • ライセンス費用のみ。推論回数・ユーザー数による追加課金なし
  • 全処理がデバイス内で完結。データが外部に出ない
  • ailia Tokenizer / Speech / LLM / TrackerがUnity・Kotlin・C++に対応

ONNX Runtimeとの比較

汎用ランタイムではなく、
特定用途で超えられる理由

ライブラリをすべて内製しているため、特定デバイス・特定モデルに特化した最適化が可能です。Vulkan・Metal直接実装やモデル特化チューニングなど、汎用設計では対応しにくい領域をカバーします。

アイリアSDK ONNX Runtime
Vulkan・Metal独自実装によるGPU推論
Android/Windows/Linux（Vulkan）、iOS/macOS（Metal直接実装）
音声認識・音声合成・LLM・トークナイザー・物体追跡ライブラリ
ailia Speech / Voice / LLM / Tokenizer / Tracker — Unity・Flutter・C++対応
400以上の公式モデルライブラリ
動作検証済みモデルとサンプルコードをすぐに利用可能
Non-OS / RTOS対応
内製ライブラリによる任意OSへの対応
Unity公式バインディング＋モデル集
ゲーム・XR向けにモデルを選んで即デプロイ
モデル特化最適化
内製ライブラリによる特定モデル・デバイスへの最適化
